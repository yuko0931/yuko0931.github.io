对于seo（搜索引擎优化）的认识

seo

搜索引擎优化

条件

  • 多页面 ==》 spider抓取
  • 页面含有蜘蛛抓取的内容
  • title、描述、关键词

vue的解决方案

预渲染

需要使用相关vue插件以及对应配置

  • prerender-spa-plugin：对路由的页面进行预渲染
  • vue-meta-info：设置页面的title、描述、关键词

缺点

  1. 如果较多详情页需要设置seo，工作量较大，不适合
  2. 动态的改变title、描述、关键词也不行

适用

某几个页面做seo

SSR

  2. 将打包后的文件拷贝到服务器中（有node环境
  • nuxt.config

  • package.json
  • static
  1. 服务器：npm install
  2. 运行npm run start
